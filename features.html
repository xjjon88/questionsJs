Features

All our latest and upcoming features
Like / Dislike
Separate like / dislike counters show the level of consensus towards each question.
Hashtags
Add hashtagged terms to posts to create suggested searches for other users.
Color Scheme / Night Mode
Varying color schemes allow users to decide based on personal preference and viewing environment.
Extra Admin Functionality
Rather than delete, hide questions that are inappropriate or no longer needed. Hiding provides and layer of protection against accidental deletions. Pin important posts to the top of the page to increase visibility.
Reply / Comment
Allow user to post their answer and comment to the question.
Question Sorting
Sorting the questions by different methods, such as highest like, highest dislike and newest question.
Profanity Filter
Filter out the bad words.
Classroom Display mode
Let the user to focus on one of the questions only.
UI Improvement
Change the design of the question room.
